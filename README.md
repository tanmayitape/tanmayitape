## Flagship Project
<p align="center">
  <a href="https://github.com/tanmayitape/f1-race-winner-prediction">
    <img src="https://github-readme-stats.vercel.app/api/pin/?username=tanmayitape&repo=f1-race-winner-prediction&show_owner=true" />
  </a>
</p>

**Predicting Formula 1 Race Winners Using Pre-Race Machine Learning (2017–2024)**  
Time-aware modelling (2017–2022 → 2023–2024) on a highly imbalanced “one winner per race” setup. Compared Logistic Regression, Random Forest, and XGBoost.  
**Outcome:** XGBoost delivered the strongest winner-class balance (Precision/Recall/F1) with better-calibrated probabilities, showing F1 is predictable in structure, not fully in execution.
